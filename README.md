# iFriend Contact Manager

In the early days of my coding journey, I introduced one of my first GUI applications â€“ a Contact Manager. Fueled by JFrame, this enduring user-friendly interface simplifies contact management with seamless options for adding, deleting, and editing.

---

## Features

### Contact Management

- **Add, Delete, Update**: Easily manage contacts.

### Data Validation

- **10-Digit Numbers, First Digit is 0, Birthday Validation**: Ensure accurate contact details.

### Sorting Options

- **Sort by Name, Birthday, Salary**: Organize contacts effortlessly.

### Visualization

- **Table View**: Display contacts in a structured table.

---


## Getting Started

To run the Contact Manager:

1. Clone the repository.
2. Open the project in your preferred Java IDE.
3. Navigate to the `HomePage.java` file.
4. Run the `HomePage.java` file to launch the Contact Manager.

Note: Ensure you open the entire project folder, not just the individual file, to ensure proper execution of the code.
